Add a UTM & URL builder to your site for GA4.
 
== Description ==
 
Provide your visitors with an easy way to build perfect UTM based URLs for their marketing and landing pages for GA4 (Google Analytics 4).

== Installation ==
 
1. Upload the plugin folder to your /wp-content/plugins/ folder.
2. Go to the **Plugins** page and activate the plugin.
3. Add this shortcode where you want the form to appear: [add_ga4_utm_builder]

 
== Frequently Asked Questions ==
 
= How do I use this plugin? =
 
Add this shortcode where you want the form to appear: [add_ga4_utm_builder]
 
= How to uninstall the plugin? =
 
Simply deactivate and delete the plugin.
